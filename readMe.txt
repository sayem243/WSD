 Cypress Automation with Cucumber BDD

## Overview
This project is an automated testing suite using **Cypress** with **Cucumber BDD** for behavior-driven development. It includes feature files, step definitions, and test execution reports.

## Tech Stack
- **Cypress**: JavaScript-based end-to-end testing framework
- **Cucumber BDD**: Behavior-driven development using Gherkin syntax
- **Node.js**: Required for Cypress execution
- **Git**: Version control

## Installation

1. Clone the repository:
   https://github.com/sayem243/WSD
2. Open the project in VS code 

3. Run the code: npx cypress open

4. Select browser

5. click search feature
